# OraeCF Product Master

This file is the shared product reference for website copy, Amazon content, and packaging artwork.

Brand note: `OraeCF` is the registered brand name. `Orae` is the display name/logo used visually on the current website and carton artwork.

## Shared Claims

These claims are common to all four fans and are safe for shared carton artwork:

- Orae 3-blade ceiling fan
- Ultra-quiet DC motor
- 23 dB motor noise claim
- Up to 85% energy saving compared with traditional AC motor fans
- Remote control included
- 6 speed settings
- Timer function
- Reversible summer/winter airflow
- Modern minimalist design
- Suitable for bedrooms, living rooms, and offices

Avoid using the LED claim on shared packaging because only the two wood models include integrated lighting.
